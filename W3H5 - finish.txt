Alternatives to Google Maps API 
Microsoft Maps API - Microsoft Maps uses the Bing search engine along with ASP.net, SQL Server, and Microsoft Azure.  There are two different services for Microsoft Maps, Basic and Enterprise.  The difference between the two are the number of billiable transactions that can be allowed on the Microsoft platform.
Osmdroid - OpenStreetMap is a Map API that is optimized for Android.  It is a GPS navigation and map application that features optional offline maps and turn by turn directions.  It is open source software project that incorporates 100% free OpenStreetMap data to deliver complete, user-friendly, mobile GPS solution.  It supports multiple languages such as Russian, German, Arabic, Spanish, and Hindi.  It is available on the Amazon App store and the Google Play Store for a minimal price.  The source code can be found on GitHub.
